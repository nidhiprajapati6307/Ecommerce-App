# Ecommerce-App
Ecommerce App is a scalable e-commerce mobile application built using Flutter. The app follows the MVVM (Model-View-ViewModel) architecture and integrates REST APIs for authentication and product management.
🚀 Features
🔐 User Authentication (API-based Login & Signup)
🛍️ Browse Products
📦 Product Details Screen
🛒 Add to Cart Functionality
❤️ Wishlist (Optional)
🔄 API Integration (RESTful services)
📱 Responsive UI (Android & iOS)
⚡ State Management (Bloc)

🧠 Architecture – MVVM

This project follows the MVVM architecture, which helps in maintaining clean and scalable code.

🔹 Model
Represents data layer
Handles API response models and data structures
🔹 View
UI layer (Screens & Widgets)
Displays data to users
🔹 ViewModel
Acts as a bridge between Model and View
Handles business logic and API calls
Notifies UI on data changes
🔌 API Integration
REST APIs are used for:
User login & registration
Fetching product list
Managing cart data
Uses http or dio package for network calls
🛠️ Tech Stack
Flutter & Dart
State Management: GetX / Provider
Architecture: MVVM
API Handling: REST APIs
JSON Parsing
